Documentation:

The following examples assume that you are running the Radio API locally on port 5000. If that is the case, then both the "Try it" examples and the curl sample commands should work.

Play a stream:

To play a stream, post a form field called streamurl with a url encoded value containing the stream URL to /api/stream, eg:
curl --data "streamurl=your_encoded_url" http://localhost:5000/api/stream

On success, the result will be a status code 200 with the text OK.

Stop playing:

To stop playing, post a form field called streamurl with a value of stop to /api/stream, eg:
curl --data "streamurl=stop" http://localhost:5000/api/stream

On success, the result will be a status code 200 with the text OK.

Increase volume:

To increase the volume, post a form field called vol with a value of up to /api/incvol, eg:
curl --data "vol=up" http://localhost:5000/api/incvol

On success, the result will be a status code 200 with the text OK.

Try it:

Reduce volume:

To reduce the volume, post a form field called vol with a value of dn to /api/incvol, eg:
curl --data "vol=dn" http://localhost:5000/api/incvol

On success, the result will be a status code 200 with the text OK.

Try it:

Set volume:

To set a specific volume, post a form field called vol with a value in the range 0 - 20 to /api/setvol, eg:
curl --data "vol=4" http://localhost:5000/api/setvol

On success, the result will be a status code 200 with the text OK.
